📸 Tema abordado

O site simula uma rede social simples, onde usuários podem visualizar postagens, acessar perfis e interagir com um ambiente semelhante às plataformas sociais modernas.

🎯 Objetivo do projeto

Este projeto foi desenvolvido com foco no aprendizado de HTML, explorando a criação de múltiplas páginas e a navegação entre elas, além da estruturação de conteúdos típicos de uma rede social.

🚀 Funcionalidades atuais

Navegação entre páginas

Exibição de posts no feed

Página de perfil com informações do usuário

Formulários de login e cadastro

💡 Melhorias futuras

Adicionar estilização com CSS

Implementar interações com JavaScript

Sistema de curtidas e comentários

Upload de imagens nos posts

Simulação de autenticação de usuários
